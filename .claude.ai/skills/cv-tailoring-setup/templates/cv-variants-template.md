# CV Variants

This is the complete inventory of CV variants bundled with this skill. They are in the `cv-variants/` directory within the skill folder. Read all of them before starting any tailoring work.

To find the skill's directory, look for the path where you loaded this file from. The CV variants will be in a sibling directory called `cv-variants/`. For example, if this file is at `/mnt/skills/user/cv-tailoring/references/cv-variants.md`, the variants are at `/mnt/skills/user/cv-tailoring/cv-variants/`.

## How to use the variants

When reading the variants, pay attention to:

- How each role is described differently across variants. The same position may emphasise different deliverables depending on the target role.
- Technical details that appear in only one or two variants. These are often the most valuable for tailoring because they represent specific experience the base CV may not surface.
- Project descriptions and outcomes. Some variants include project narratives that others omit.

After reading all variants, identify the best starting base for the target role and note which specific content from other variants should be pulled in.
