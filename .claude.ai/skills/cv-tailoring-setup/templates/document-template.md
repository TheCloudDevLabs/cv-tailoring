# Document Template

This reference defines the formatting specification for CV and supporting documents. When generating .docx files, read the docx skill at `/mnt/skills/public/docx/SKILL.md` for the technical approach, then apply the formatting defined here.

## CV layout

The CV uses an A4 page with a two-column layout:

- Left column (approximately 70% width): main content
- Right column (approximately 30% width): sidebar with contact details, skills, and links

The layout is implemented as a table with invisible borders.

### Page properties

- Page size: A4 (11906 x 16838 DXA)
- Margins: top 720 DXA (0.5 inch), bottom 720 DXA, left 720 DXA, right 720 DXA
- Font: Calibri throughout (fallback: Arial)
- Base font size: 10pt for body text

### Header section

Spans both columns at the top:

- Name: Calibri, 20pt, bold, dark colour (#2E3440)
- Title/tagline: Calibri, 12pt, regular weight, below the name

### Left column sections

In order from top to bottom:

**Summary**
Heading: "Summary", 12pt bold. Body text: 10pt, paragraph format. Two to four paragraphs covering career overview, domain focus, and key capabilities relevant to the target role.

**Experience**
Heading: "Experience", 12pt bold. A bullet list of technical capabilities and experience areas relevant to the target role. Keep entries concise (one to two lines each).

**Employment History**
Heading: "Employment History", 12pt bold. Each role entry:

- Role title and employer: bold, 10pt (format: "Role Title, Employer, Location")
- Date range: regular weight, 10pt
- Role description: 10pt. Use a brief intro paragraph (one to two sentences of context) followed by bullet points for specific deliverables. Do not use narrative paragraphs for the deliverables.

**Certifications**
Heading: "Certifications", 12pt bold. Simple text list.

### Right column sections

**Details**
Contact information: mobile, email. Plain text, 9pt.

**Links**
LinkedIn URL as a hyperlink. GitHub URL if provided. Personal website if provided. 9pt.

**Skills**
Skill labels relevant to the target role. 9pt, plain text labels only — do not attempt visual skill bars.

### Colours

- Headings: #2E3440 (dark grey, almost black)
- Body text: #2E3440
- Sidebar text: #4C566A (slightly lighter grey)
- Sidebar heading labels: #2E3440, bold
- No background colours, coloured borders, or decorative elements

### Spacing

- Space before headings: 240 DXA (approximately 12pt)
- Space after headings: 120 DXA (approximately 6pt)
- Space between paragraphs: 120 DXA
- Line spacing: single (1.0) or 1.15
- Space between employment history entries: 240 DXA

## Supporting document layout

Cover letters, selection criteria responses, and pitch documents use a simpler single-column layout.

### Page properties

- Page size: A4
- Margins: top 1440 DXA (1 inch), bottom 1440 DXA, left 1440 DXA, right 1440 DXA
- Font: Calibri throughout
- Base font size: 11pt for body text

### Header

- Name: bold, 14pt
- Contact line: regular, 10pt (email, phone, location, separated by pipes)
- Security clearance line: bold, 10pt (if applicable)
- GitHub or website line: 10pt (if provided)

### Body

- Section headings (for selection criteria): bold, 11pt
- Body text: 11pt, paragraph format
- No bullet lists unless specifically appropriate
- Single spacing with 120 DXA between paragraphs

### Cover letter specifics

- Date at top
- Addressee if known, otherwise "Dear Hiring Manager"
- Three to four paragraphs maximum
- Sign-off with name

### Selection criteria specifics

- Each criterion as a separate section with the criterion text as a heading
- Narrative response using specific project examples
- Each response self-contained
- Typical length: three to five paragraphs per criterion
